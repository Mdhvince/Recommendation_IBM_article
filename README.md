
# Recommendation engine for article on IBM Plateform

Analyze the interactions that users have with articles on the IBM Watson Studio platform, and make recommendations to them about new articles you think they will like.

## File Description
- rec_eng_ibm.ipynb: data cleaning and nlp + recommendations
- user-item-interactions.csv: interactions of users on articles
- articles_community: all articles + descriptions on ibm plateform
- recommender_finctions.py / recommender.py: My own Recommender module, see more <a href='https://github.com/Mdhvince/Recommendation_engine_module'> here </a>
